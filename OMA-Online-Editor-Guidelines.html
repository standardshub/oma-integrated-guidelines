How to Use this Document

 OMA has developed an online/offline editor (OMA Editor) to support the development of OMA Technical Documents.

 The document is constructed using plain HTML, without any styling and it can be rendered by one of these two browsers:

  • Chrome, or
  • Firefox

At the time of rendering the document, the style is applied by the browser based on a set of specific styling rules (ReSpec).

Sections Constructed Automatically

The following sections and content are constructed automatically by the browser at the time of rendering the document. These sections or content doesn't exist inside of the document:

  • Legal Disclaimer,
  • Table of Contents,
  • Table of Figures,
  • Tables of Tables,
  • Section Numbers,
  • Tables and Figures numbers,
  • References,
    • Normative, and
    • Informative
  • Appendix numbers

ID's

  • Any html element MAY have an ID.
  • ID's are used to link back to that element from inside or outside of the document.
ID's
Operation Description Problems
Insert ID's

Follow these steps to insert an ID:

  1. Place the cursor or select the element that you want to add an ID. (see below)
  2. Click in "Edit ID" button on the Editor Menu.
  3. Input in the pop up window the value of the ID. (see rules)

Rules for creating ID's

  • When creating an ID follow this basic rules.
    • DO NOT use capital letters.
    • Separate every word with the symbol "-".
    • DO NOT use special characters such as: "[ ', !, &, %, etc]
  • Abbreviations
    • Place the cursor next to the Abbreviation.
    • ID starts with the letter "a-" for abbreviation, followed by the specific ID: e.g. ID in an abbreviation, e.g.: "a-oma".
  • Definitions
    • Place the cursor next to the Definition.
    • ID starts with the letter "d-" for definition, followed by the specific ID: e.g. ID in a definition, e.g.: "d-dm".
  • Figures
    • Select the Figure.
    • ID starts with the letter "i-" for figures, followed by the specific ID: e.g. ID in a definition, e.g.: "i-brick-wall".
  • Lists
    • Place the cursor in the List.
    • ID starts with the letter "l-" for lists, followed by the specific ID: e.g. ID in a list, e.g.: "l-architecture".
  • Paragraphs
    • Place the cursor inside of the Paragraph.
    • ID starts with the letter "p-" for paragraphs, followed by the specific ID, e.g.: "p-section-and-appendix".
  • Preformatted Text/Code
    • Place the cursor inside of a Preformatted Text Box.
    • ID starts with the letters "b-" for preformatted text, followed by the specific ID, e.g.:  "b-text-box"
  • Sections, Subsections, Appendices, References
    • DO NOT NEED to INSERT an ID
    • To link to one of the elements above just input a hyperlink where the URL is url="#section-title".
      • Follow the basic above rules to construct the 'section-title'.
