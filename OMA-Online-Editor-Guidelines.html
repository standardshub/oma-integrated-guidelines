How to Use this Document

 OMA has developed an online/offline editor (OMA Editor) to support the development of OMA Technical Documents.

 The document is constructed using plain HTML, without any styling and it can be rendered by one of these two browsers:

  • Chrome, or
  • Firefox

At the time of rendering the document, the style is applied by the browser based on a set of specific styling rules (ReSpec).

Sections Constructed Automatically

The following sections and content are constructed automatically by the browser at the time of rendering the document. These sections or content doesn't exist inside of the document:

  • Legal Disclaimer,
  • Table of Contents,
  • Table of Figures,
  • Tables of Tables,
  • Section Numbers,
  • Tables and Figures numbers,
  • References,
    • Normative, and
    • Informative
  • Appendix numbers

Sections

  • This section describes how sections, subsections and appendix sections work.

Sections

All the section functions are located in the "Sections" button.

Sections
Operation Description Problems
"Add Section"

Follow these steps:

  1. Place the cursor where you want to add the section.
  2. Click "Sections"/"Add Section" button on the Editor Menu.

Note: if the cursor was placed inside of a section the new section will be inside of this section, (nested section).

If you want to create a new section outside of any section, see below.

"Add Section After"

Follow these steps to add a section at the same level than another section.

  1. Place the cursor, inside section level X, 
  2. Click "Sections" / "Add Section After" button on the Editor Menu.
    • The Editor will insert the new section just after section level x, at the same level

Note: If you want to add a nested section see above description.
"Delete Section"

Follow these steps:

  1. Place the cursor inside of the section or subsection to delete.
  2. Click "Sections" / "Delete Section" button on the Editor Menu.
"Move Section Up"

Follow these steps:

  1. Place the cursor inside of the section or subsection to move up.
  2. Click "Sections" / "Move Section Up" button on the Editor Menu.
  3. The Editor will move the selected section one section up.
"Move Section Down"  

Follow these steps:

  1. Place the cursor inside of the section or subsection to move down.
  2. Click "Sections" / "Move Section Down" button on the Editor Menu.
  3. The Editor will move the selected section one section down.
"Unindent Section"  

Follow these steps:

  1. Place the cursor inside of the section or subsection to be unnested.
  2. Click "Sections" / "Unindent Section" button on the Editor Menu.
  3. The Editor will move the selected section outside of the nested section and one section down.
Hyperlink to an existent section

Follow these steps:

  1. There is no need to create an ID or anchor to link to a section, subsection or appendix.
    • The system is able to identify, by default, all the section titles without the need to introduce an "ID" or "anchor".

To link back from inside or outside of the document to a particular section follow these steps:

  1. Select the word that you want to link to the section hat you can use as a link.
  2. Click in "Insert/Edit Link" button on the Editor Menu.

Insert the following values:

  1. URL
    • Insert # followed by the section name. Rules to create the section name:
      • use the same title as it appears in the section to link to.
      • use lowercase only.
      • use (-) to separate words.
      • eliminate any special character e.g. "(", "-", etc.
      • e.g. #terminology-and-conventions
  1. Text to Display
    • This is the word/statement that contains the link to click
  2. Title
    • This content is displayed when hovering over the word/statement
  3. Anchor
    • none
  4. Target
    • none
Find Section ID

 Follow up these steps to find the ID for a section:

  1. Place the cursor over the section title that you want to obtain the Section ID.
  2. Look at the URL displayed in the bottom-left-hand corner of the screen.
  3. Identify the last part of the URL starting with "#", this is the section ID that you can use inside of hyperlink to jump to that section., e.g.: #how-to-use-this-document.

ID's

  • Any html element MAY have an ID.
  • ID's are used to link back to that element from inside or outside of the document using a hyperlink element.
ID's
Operation Description Problems
Insert ID's

Follow these steps to insert an ID:

  1. Place the cursor or select the element that you want to add an ID. (see below)
  2. Click in "Edit ID" button on the Editor Menu.
  3. Input in the pop up window the value of the ID. (see rules)

Rules for creating ID's

  • When creating an ID follow this basic rules.
    • DO NOT use capital letters.
    • Separate every word with the symbol "-".
    • DO NOT use special characters such as: "[ ', !, &, %, etc]
  • Abbreviations
    • Place the cursor next to the Abbreviation.
    • ID starts with the letter "a-" for abbreviation, followed by the specific ID: e.g. ID in an abbreviation, e.g.: "a-oma".
  • Definitions
    • Place the cursor next to the Definition.
    • ID starts with the letter "d-" for definition, followed by the specific ID: e.g. ID in a definition, e.g.: "d-dm".
  • Figures
    • Select the Figure.
    • ID starts with the letter "i-" for figures, followed by the specific ID: e.g. ID in a definition, e.g.: "i-brick-wall".
  • Lists
    • Place the cursor in the List.
    • ID starts with the letter "l-" for lists, followed by the specific ID: e.g. ID in a list, e.g.: "l-architecture".
  • Paragraphs
    • Place the cursor inside of the Paragraph.
    • ID starts with the letter "p-" for paragraphs, followed by the specific ID, e.g.: "p-section-and-appendix".
  • Preformatted Text/Code
    • Place the cursor inside of a Preformatted Text Box.
    • ID starts with the letters "b-" for preformatted text, followed by the specific ID, e.g.:  "b-text-box"
  • Sections, Subsections, Appendices, References
    • DO NOT NEED to INSERT an ID
    • To link to one of the elements above just input a hyperlink where the URL is url="#section-title".
      • Follow the basic above rules to construct the 'section-title'.
  • Tables
    • Select the Table.
    • ID starts with the letter "t-" for table, followed by the specific ID, e.g. : ID for a table: "t-table-constructors".
Update / Edit ID's

Follow these steps:

  1. Select the constructor or paragraph that you want to see its "ID".
  2. Click in "Edit ID" button on the Editor Menu.
  3. A window will pop up showing the value of the element ID.

Please note:

  • To view an existing element "ID" this MUST have been previously inserted in the element. See Insert ID's.


  • Tables:
  1. Select the table, and
  2. Click on "Insert Table" button on the Editor Menu to see the table ID.

                  or

  1. Select the table, and
