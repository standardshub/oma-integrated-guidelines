  • The addition of a new user requires to input the following information:
    • Name
      • Name of the user.
    • Email
      • User email address.
    • Role
      • Admin
        • Admin access to the application.
      • Staff
        • Full write access to all the functions in the application.
      • Officers
        • Partical write access to the functions in the application.
      • Editors
        • A more restrictive write access to the functions inthe application.
      • Members
        • Mainly read access only to:
          • Work Program
            • Ability to create and delete own Change Requests.
          • Change Request
            • Ability to Witdrawn or create a new revision of a document submitted by the user.
    • Password
      • Input by the staff at the time of creating the user account.
    • Password Confirmation
